## 网络线路与带宽情况

Terabit的网络分两档：

**标准线路（Standard Transit）：** 使用Cogent + HE（Hurricane Electric）的线路。Cogent是老牌Tier 1运营商，HE则是全球IPv6流量最大的网络运营商之一。这两家的线路质量中规中矩，适合大部分场景。

**高级线路（Premium Transit）：** 采用CDN77和DataPacket的线路。这两个在欧美地区的表现相对更好，延迟更低，丢包率也更稳定一些。当然，高级线路需要额外付费。

每个VPS默认配一个IPv4地址（/32），还送一个/64的IPv6子网。如果需要更多IP地址，可以联系客服购买。

所有套餐都包含99.9%的SLA保证，这个在业内算是标准配置了。实际稳定性怎么样，还得看后续使用情况。毕竟是新商家，长期表现还需要时间验证。
